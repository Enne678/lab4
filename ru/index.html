О проекте

Это демонстрационный проект, показывающий возможности автоматизированного развертывания веб-приложений на GitHub Pages.

Возможности

  • Мультиязычность (русский и английский)
  • Автоматическое развертывание через GitHub Actions
  • Версионирование релизов
  • Доступность предыдущих версий

Технологии

HTML5, CSS3, JavaScript, GitHub Actions, GitHub Pages
